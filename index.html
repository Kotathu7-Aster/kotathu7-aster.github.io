　 24/04/18 　

リンク集を見直しました。インターネットやプログラミングにおいてセキュリティ向上が具体的な動きになってきました。スパムメール対策やパスワードレス認証などの動きのなかで、開発者も利用者もアプリやオンラインサービスへの向き合い方に急激な変容を迫られています。

また、本サイトのオンラインソフトを掲載いただいたこともあるDOS/V POWER REPORTが2024年冬号で休刊となるなど、時代の変化を象徴する出来事もありました。

授業や入試に情報教育が標準装備された世代が作るインターネット世界に期待しながら、それに付いていけるか心配している今日この頃です。


　 23/03/13 　

本日Easter Ver.0.97 を公開しました。Windows 10 / 11 にてダークモード設定時に機器選択画面を暗い配色で表示する機能を追加するとともに、安全性を向上させる修正を行いました。ダウンロードページはこちらです。

Easterの初版公開から約12年半が経過し、PCの拡張方法やUSBの実装が大きく変わったことを改めて実感しています。幅広いWindowsバージョンや機器に対応できるよう心がけていますが、PCMCIAについては、最新のWindowsが動作する環境が用意できないため、Windows 7(32bit)上での検証にとどまっています。ご了承ください。

マスクの着脱が個人の判断になり、アフターコロナへの動きが本格化してきました。皆様のご健康をお祈り申し上げます。


　 22/03/23 　

本サイトは22年目に突入しています。アクセスカウンターも10万2766回（2022年3月23日18:30現在）を表示しており、多くの方に支えられていることを改めて実感しています。

最後の更新から5年以上が経過し、特に2020年から続く新型コロナウイルス感染症の拡大を受けて、ICT 技術を活用したリモート会議が一般的なものになるなど、急激な社会の変化が生じています。
IoT、ビッグデータ解析といったプログラミングの潮流から、Python や Kotlin が隆盛を誇っています。また、Android アプリの実行や、WSL2によるLinux環境の構築をサポートする Windows 11 が発売され、Win32 API を用いたネイティブアプリの個人開発はますます旧世代の営みとなりました。
30年近くにわたりバイナリ互換を保つ Windows と、20年以上にわたりホームページサービスを継続している Vector の懐の深さに頼ってばかりですが、今後も何らかの形でホームページでのプログラム公開を維持していきたいと思いを新たにしています。


　 17/01/23 　

Easter及びPortierの最新版を公開しました。

Easter (Ver 0.96)については、不具合の修正に加え、新たに Windows 8.1 / 10 環境での Per-Monitor DPI に対応しました。
Portier (Ver 2.21)は UAC の動作や 64bit 環境への対応を中心に、最新の Windows 10 までの環境での互換性を高めました。

Portierは初版公開から10年を超え、Easterは6年を超えました。どちらのソフトもユーザー様に大切に使っていただき、うれしく思っています。


　 17/01/14 　

リンク集を修正しました。

当サイトがオープンから約5年間使用していたOCN PageONサービスをはじめ、各社のホームページ公開サービスの終了に伴い、サイトリストが寂しくなってきました。各種の環境の変化を身をもって感じる今日この頃です。

主なソフトウェアのWindows10対応状況は以下の通りです。
Easterについては、Ver 0.95以降が問題なく動作します。
Portierについては設定プログラムの一部でエラーがありましたが、2017/1/23公開のVer 2.21で修正しています。

今後もご支援のほどお願い申し上げます。


　 15/06/27 　

久しぶりの更新となりました。リンク集を修正しています。

Windows 10 リリースが秒読み段階となり、タブレット・MVNOの隆盛とともに情報機器のプラットフォームも激変しています。USBについても、新規格対応PCが続々と出てきていますが、EasterやPortierはこれまで通り動くようサポートいたします。
当サイトは間もなく15周年を迎えます。これまでのご支援に感謝いたします。


　 13/12/07 　

関係の皆様にはずいぶんお待たせすることになってしまいましたが、本日Easterの最新版Ver.0.95をアップしました。Windows8/8.1に正式対応したほか、これまで懸案だった起動失敗の不具合についても解消されています。

さて、あっという間に2013年も最後の月です。このところ旧来型のWindows APIプログラミングの比重が減り、私生活でもスマートフォンやタブレットを当たり前に使うようになって、環境の変化を肌で感じているところです。来年2014年はXPサポート終了の年、フリーソフト作者にとっても一区切りの年になるのではないかと考えています。
