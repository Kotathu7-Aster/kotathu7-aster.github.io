上位ソフト「Portier 」を公開しました。Portierへの移行をお願いします(2010年10月15日)


ソフトの特徴

アイコンをクリックするだけで、光学ドライブのトレイを出したり、しまったりするソフトです。
イジェクトボタン感覚で光学ドライブのトレイを操作できます。
  • ドライブの状態を自動判断し、開け閉めします。
  • 一般のソフトと違い、常駐しません。
  • コマンドラインだけでなく、便利な専用設定プログラムによる動作のカスタマイズもサポート。
  • ドライブが二台以上接続されている場合にも対応。
  • 動作をカスタマイズしても、起動するだけの簡単操作は変わりません。
  • レジストリを使わないので、安心してお試しいただけます。

Windows 95でのご使用について

本体ファイルはこれまでどおりご使用可能ですが、ヘルプファイル(ocdhelp.chm)の閲覧および専用設定プログラム(ocdconf.exe)のご利用にはInternet Explorer 4.0 以上がインストールされている環境が必要です。
専用設定プログラムはフォントにMS UI Gothicを使用していますので、別途インストールしてください。

動作確認情報

デスクトップ用トレイ式CD/DVDドライブにおいては、トレイの収納、排出ともにほぼ100％動作します。
スロットインタイプおよび薄型ドライブ、仮想ドライブなど、ディスク（およびトレイ）のローディングがソフト側から操作できないものは取り出しのみ可能です。

作者が動作を確認しているドライブの一覧については、 こちらをご覧ください。

ダウンロードの前に

  • 添付文書およびヘルプをよく読んだ上でご使用ください。
  • 本ソフトウェアはフリーソフトです。

インストール方法

ダウンロードしたファイルを同一のディレクトリに解凍してご使用下さい。
本ソフトがレジストリやシステムファイルに変更を加えることはありません。
また、ランタイム等は必要ありません。

以前のバージョンがインストールされている場合

今回のバージョンは、構成ファイルがすべて更新されていますので、アップデート用の差分アーカイブはありません。
アップデートの場合も下のリンクからダウンロードしてください。

旧バージョンの構成ファイルをバックアップのうえ、すべてのファイルを上書きすると更新されます。
なお、旧バージョンのocd.iniおよびオプションスイッチ（Ver2.0以降）はそのまま使うことができます。詳しくは、添付文書およびヘルプファイルをご覧ください。
